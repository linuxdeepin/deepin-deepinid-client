# deepin-deepinid-client

深度云账户和云同步。

## 依赖

deepin-deepinid-client 基于 QWebEngine 开发，可以使用以下的命令安装构建依赖:

```bash
#For release based on debian
sudo apt-get build-dep deepin-deepinid-client
```

## 构建

`make`

## 安装

`sudo make install`
